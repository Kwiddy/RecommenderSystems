KEY INFORMATION
- This recommender has been developed using Python 3.8.3 and uses a CLI

RUNNING THE PROGRAM
- The recommender system should be run by first running:
  python main.py

- Upon starting the system, there will be the option to input a single character 
  indicating your choice from the menu. This style of input is used throughout.
  For example, you first may input the ID of an existing character by entering "E",
  select from a list of pre-chosen IDs by entering "S", or create a new user by 
  entering "N".
